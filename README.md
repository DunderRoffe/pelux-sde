PELUX-SDE
=========

Software Development Environment for the PELUX distro.

Maintained at [https://github.com/Pelagicore/PELUX-SDE](https://github.com/Pelagicore/PELUX-SDE)

License: MPLv2.0

Build
-----
Dependencies:
* Vagrant
* VirtualBox

```bash
vagrant up
```

Testing
-------
Dependencies:
* pytest

```bash
test/run-tests.sh
```
